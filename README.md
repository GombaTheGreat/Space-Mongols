# Space Mongols: Karakorum Drift

Repository for the **Space Mongols: Karakorum Drift** SillyTavern character card pack.

This project contains the static portfolio site and supporting assets for a shipboard space opera setting aboard the Karakorum Drift, including character roster previews, setting overview, artwork, the opening theme, and the Space Shagai mini-game.

## Portfolio page

The public portfolio page is here:

https://gombathegreat.github.io/Space-Mongols/

If the link is fresh and GitHub is still deploying, give GitHub Pages a few minutes to finish building.

## Release contents

The release pack includes:

- 10 SillyTavern character cards
- 1 Storyteller GM card
- RC2 lorebook
- License file
- Public portfolio page assets
